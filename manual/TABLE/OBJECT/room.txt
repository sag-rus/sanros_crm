ROOM - таблица номеров объекта

id - уникальный идентификатор
name - название номера
active - архив (0 - не в архиве, 1 - в архиве)
id_obj - объект
id_comfort - перечисления удобств (id через запятую)
note - примечание к номеру (корпус, этаж...)
people - вместимость номера
housing - id корпуса
service_info  -
main_place - количество основных мест
add_place - количество доп мест
id_best_comfort - перечисления удобств с иконкой
square - площадь
food - питание
accessible_places - квота мест
price_places - цены для квоты мест
description - описание
