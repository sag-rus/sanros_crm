DIRECTION_OBJECT - направления (регионы, федеральные округа)

id - id
name - название
id_reg - id региона (указывается для городов-курортов в Крыму, КавМинводах)
id_country - id страны (указывается для федерального округа)
description - описание
name_rod - название в род падеже
meta_desc - мета описание для сайта
