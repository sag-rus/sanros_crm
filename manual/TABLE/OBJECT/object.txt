OBJECT - таблица объектов

id - уникальный идентификатор
name - краткое имя
id_reg - регион
active - архив (0 - не в архие, 1 - в архиве)
id_profile - перечисления профилей лечения (id через запятую)
id_methods - перечисления методов лечения (id через запятую)
id_infa - перечисления инфраструктуры (id через запятую)
id_disease -
id_children - перечисления детей (id через запятую)
note - примечание
type - тип объекта (cанаторий, отель, база отдыха)
full_name - полное название объекта
address - адрес
telephone - JSON массив с контактными телефонами
email - JSON массив с email
id_tour - ID туроператора по умолчанию
city - город
description - описание
service_info - услуги, массив
arrival - время заезда
leaving - время выезда
add_one_day - как считает объект - днями или сутками
regular_com - комиссия по умолчанию
latitude longitude - координаты
similar - похожие объекты
direction - направление
reward - вознаграждение
description_check - описание от объекта из личного кабинета (проходит нашу модерацию)
status - статус изменения (изменял ли объект информацию из ЛК)
1C_code - код из 1С
inn - ИНН
nomenclature - номенклатура 1С
1C_full_name - полное название из 1С
url_name - ссылка (санатории-россии.рф)
website - официальный сайт
bank_login - логин альфа банка (для разделения платежа)
medical_factors - медицинские факторы
check_places - статус выгрузки квоты (1 - квота travelline, 2 - квота САНАТА)
note_reward - примечание к вознаграждению
