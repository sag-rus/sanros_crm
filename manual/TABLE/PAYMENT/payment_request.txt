payment_request - таблица запроса оплаты банковской картой (через ЛК туриста)

id - id
bid - номер заявки
bid_pay - номер заявки плюс номер попытки оплатить картой (нельзя создавать одинаковые заявки в банке)
time - время
type - тип оплаты (1 - оплата, 2 - предоплата)
sum - сумма платежа
pay_method - способ оплаты (5 - банковская карта)
bank_com - комиссия банка в процентах
order_id - уникальный номер заказа в системе банка
status - статус платежа
booking - номер заявки от санатория
