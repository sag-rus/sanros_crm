TIME_PAYMENT - таблица сроков оплаты (при установке предоплаты - можно сделать предоплату через личный кабинет на указанную сумму)

id - id
id_schet - id заявки
type - тип (1 - полная оплата, 2 - предоплата)
percent - процент (не используется)
date - дата, до какого оплатить
sum - сумма (для предоплаты)
