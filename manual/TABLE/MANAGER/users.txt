USERS - таблица пользователей

id - уникальный идентификатор
ligin - логин
password - пароль
rights - id прав
dostup - доступ (1 - есть доступ, 0 - нет доступа)
photo - ссылка на фото
name - имя
surname - фамилия
telephone - телефон
email - эл.почта
note - примечание
date_add - дата регистрации
date_last_in - дата последней авторизации
class - класс ( 1 - менеджер)
id_group - id группы
office - id офиса
chat_status - статус в чатике
update_system - статус для обновления странички (1 - нужно обновиться)
