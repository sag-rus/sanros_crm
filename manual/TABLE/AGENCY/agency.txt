AGENCY - таблица агенств

id - уникальный идентификатор
name - полное название
short_name - краткое название
telephone - телефон
email - эл.почка
address - адрес агенства
note - примечание
fax - факс
icq - аська
skype - скайп
website - сайт агенства
legal_address - юридический адрес
ag_contract - агенский договор
inn - ИНН
kpp - КПП
bik - БИК
rs - расчетный счет
ks - кореспонденский счет
bank - Банк
active - статус (0 - действующий, 1 - в архиве)
login - логин для входа в ЛК
password  - пароль (md5)
present - представитель (договор)
post - должность представителя (договор)
doc - документ (договор)
module - хэш для модуля
type_com - тип комиссии - (повышенный, обычный)
questionary - опрос
module_email - email для отправки уведомлений с заявок по модулю
1C_code - код в 1С (синхронизация)
