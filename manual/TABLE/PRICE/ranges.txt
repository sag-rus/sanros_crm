RANGES - таблица диапазонов дат для цен

id - уникальный идентификатор
name - название
id_obj - id объекта
type - тип цены (за номер, за чел/сутки, за дом, за заезд)
active - архив (1 - в архиве, 0 - не в архиве)
place - id места (взрослое, детское, мать и дитя)
id_date - id интервала дат
counter - сортировка (для удобства менеджерам)
rate_plan - id тарифного плана
