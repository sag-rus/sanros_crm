RATE_PLAN - таблица тарифов объекта

id - id
name - название
object - id объекта
status - статус (1 - действителен)
description - описание
food - питание тарифа
days - минимальное количество дней

У каждого объекта по умолчанию тариф №1 - Основной.
Если добавлять другие тарифы к объекту - тариф №1 Основной исчезает
