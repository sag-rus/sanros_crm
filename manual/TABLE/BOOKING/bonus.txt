BONUS - таблица бонусов клиента

id - уникальный идентификатор
date - дата добавления
turist - id туриста
sum - количество бонусов
schet - id заявки
active - статус (0 - в архиве, 1 - действителен)
note - примечание
type - тип (1 - за оплату путевки, 2 - перевод на другого клиента, 3 - подарочный бонус, 4 - бонус за партнерскую программу)
