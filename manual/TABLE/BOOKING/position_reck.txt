POSITION_RECK - таблица позиций счета (номера, доп услуги)

id - уникальный идентификатор
schet - id заявки
id_room - id номера
sum - цена
number - количество
note - примечание
type - тип цены (за сутки/чел, за заезд, за номер)
days - количество дней заезда
add_one_day - учитывает, как считает объект - днями или ночами
reward - вознаграждение в %
id_service - id услуги (вместо номера)
ratePlan - id тарифного плата (для квоты мест)
