HISTORY_SCHET - таблица, история действий по заявке

id - уникальный идентификатор
date - дата изменения
id_schet - id заявки
id_user - id пользователя
new_status - новый статус заявки
new_status_san - новый статус санатория
time - время изменения
note - примечание (например, пишется старые значения заявки)
